## Project Overview
An interactive Power BI dashboard analyzing Amazon Prime Video's content library, providing insights into content distribution, performance metrics, and audience trends.

The `Amazon prime Dashboard.pbix` file contains the complete interactive report.

## ğŸ¯ Goals and Key Insights

The dashboard addresses critical questions about the Amazon Prime content catalog:

1. **Content Mix**: Analyze Movies vs. TV Shows breakdown and growth trends over time.
2. **Genre Popularity**: Identify top-performing and most frequently added genres.
3. **Regional Focus**: Visualize content distribution across production countries.
4. **Rating Distribution**: Understand content volume across rating categories (e.g., TV-MA, PG-13).

## ğŸ“ˆ Dashboard Features

The report includes focused pages for different analyses:
